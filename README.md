# 🏋️ Treino Online — Personal Trainer

Site de treino personalizado com pop-up de exercícios.

## Como publicar no GitHub Pages (passo a passo)

### 1. Criar conta no GitHub
Acesse [github.com](https://github.com) e crie uma conta gratuita.

### 2. Criar um repositório novo
- Clique no botão verde **"New"** (canto superior esquerdo)
- Nome do repositório: `treino-online` (ou qualquer nome)
- Deixe como **Public**
- Clique em **"Create repository"**

### 3. Fazer upload dos arquivos
- Na página do repositório, clique em **"uploading an existing file"**
- Arraste os arquivos `index.html` e `README.md` para a área
- Clique em **"Commit changes"**

### 4. Ativar o GitHub Pages
- Vá em **Settings** (aba no topo do repositório)
- No menu lateral, clique em **Pages**
- Em "Branch", selecione **main** e clique em **Save**
- Aguarde 1-2 minutos

### 5. Acessar o site
Seu link será:
```
https://SEU-USUARIO.github.io/treino-online/
```

Pronto! Mande esse link para o aluno pelo WhatsApp. ✅

---

## Como adicionar mais alunos no futuro

Duplique o arquivo `index.html`, renomeie para `maria.html`, edite os dados e faça upload.
O link do novo aluno será: `https://SEU-USUARIO.github.io/treino-online/maria.html`
